# 삼성전자 성과급 계산기 (DS 특별경영성과급 + OPI)

2026년 5월 20일 삼성전자 노사 잠정합의서에 기반한 비공식 참고용
1인당 예상 수령액 계산기입니다.

## 프로젝트 구조

```
.
├── index.html                      # 계산기 페이지
├── data.json                       # 자사주 환산 주가 데이터 (자동 갱신)
├── scripts/
│   └── update_data.py              # Yahoo Finance 종가 fetcher
├── .github/
│   └── workflows/
│       └── update-data.yml         # 평일 18:30 KST 자동 실행
└── README.md
```

## 사용자 입력

- **사업부** — 메모리 / 공통조직 / 파운드리·LSI
- **직급** — CL1~4. CL4만 보정 계수가 실제 적용됨
- **보정 계수** — ×1.0 / ×1.2 / ×1.4 중 선택. CL4 한정. **특별경영성과급
  (부문+사업부)과 OPI 모두에 적용**. 라벨에 등급명을 표기하지 않으므로
  화면 캡처로 봐도 본인 정보를 추정하기 어렵습니다.
- **계약연봉액 + 업무 성과급** — OPI 산정 기준 (raw 값, 배수는 별도)
- **적용 연도** — 2026~2035. 임계값과 파운드리·LSI 가중치를 결정
- **DS 영업이익** — 시나리오 슬라이더 (조원 단위)
- **한계세율** — 종합소득세 한계 구간
- **자사주 환산 주가** — 기본값은 `data.json`에서 자동 갱신 (Yahoo Finance)

## 계산 공식

```
재원                = DS 영업이익 × 10.5%

메모리 부문         = 부문재원   / (N_m + 0.7×N_c + dlw×N_l)   ← X_m
메모리 사업부       = 사업부재원 / (N_m + 0.7×N_c)              ← Y_m
공통 부문           = X_m × 0.7
공통 사업부         = Y_m × 0.7
파운드리·LSI 부문   = X_m × dlw      (dlw: 2026 = 1.0, 2027~ = 0.6)
파운드리·LSI 사업부 = 0  (적자 시 사업부 재원 비참여)

OPI                 = (입력 연봉 × 50%) × 보정계수    (모든 임직원)
보정계수            = CL4 한정. 다 ×1.0 / 나 ×1.2 / 가 ×1.4
                      CL1~3 는 항상 ×1.0
세후                = 세전 × (1 − 한계세율 × 1.1)
자사주 수량         = 세후 ÷ 환산 주가
                      (1/3 즉시 매각 / 1/3 1년 제한 / 1/3 2년 제한)
분량별 가치         = 분량 주식수 × 환산 주가
분량별 세금         = 총 세금 × (분량 주식수 / 총 주식수)
```

부문·사업부 양쪽 모두 메모리를 기준으로 산출되며, 공통조직은 모든
구성요소가 메모리의 0.7배로 일관 적용됩니다.

임계값 미달 시 특별경영성과급은 0이며, OPI는 영업이익과 무관하게 연봉
기반으로 항상 지급됩니다.

## 검증

영업이익 300조·2026년·연봉 1억·보정 ×1.0 시나리오에서 부산일보 보도와
일치합니다.

| 사업부      | 부문 몫 | 사업부 몫 | OPI    | 합계   |
| :---------- | ------: | --------: | -----: | -----: |
| 메모리             |  1.83억 |    3.86억 | 0.50억 | 6.18억 |
| 공통조직           |  1.28억 |    2.70억 | 0.50억 | 4.48억 |
| 파운드리·LSI(2026) |  1.83억 |    0.00억 | 0.50억 | 2.33억 |

검증: 부문·사업부·합계 모두 공통/메모리 비율이 정확히 0.700 ✓
부산일보: 메모리 약 6억(OPI 포함). 파운드리·LSI 1.83 vs 보도 1.6은 약간
차이가 있는데, 이는 부문 재원이 가중 분배되어 메모리·파운드리가 같은
비율을 받기 때문입니다 (보도는 단순 78,000명 균등 분배 가정).

2027년부터 파운드리·LSI 부문 가중치 0.6 적용:
메모리 6.18 → 6.42, 공통 4.48 → 4.65, 파운드리·LSI 2.33 → 1.74 으로 변동.
LSI 페널티 분이 메모리·공통의 부문 몫으로 재분배되어 X_m이 1.83 → 2.07억
으로 증가하는 구조입니다.

CL4 보정 ×1.4 적용 시 모든 구성요소가 1.4배가 되어, 2026 메모리 합계는
6.18 → 8.66억으로 상승합니다.

## 유의 (반드시 참고)

이 계산기는 공개 보도를 바탕으로 한 비공식 추정 도구이며, 다음 한계가 있습니다.

- **OPI 산정 모델**은 "입력 연봉 × 50%"의 단순화된 형태입니다. 공식
  합의서의 정확한 산식이 이와 다르면 결과가 달라집니다.
- **보정 계수 적용 범위** — 본 모델은 보정 계수를 특별경영성과급
  (부문 + 사업부)과 OPI에 모두 적용합니다. 공식 합의서의 적용 범위가
  다르면 모델 수정이 필요합니다.
- **사업부 인원수**는 부산일보 보도의 추산치(메모리 28천 / 공통 30천 /
  파운드리·LSI 20천 / DS 합계 78천)이며 공식 수치가 아닙니다. 코드 상단 `HC`
  객체를 수정해 본인이 아는 값으로 바꿔 사용할 수 있습니다.
- **임계값 조건**은 특별경영성과급에만 적용됩니다. OPI 는 영업이익과
  무관하게 연봉 기반으로 지급되는 것으로 가정합니다. 공식 합의서에서
  OPI 도 임계값 조건이 걸려 있다면 모델 수정이 필요합니다.
- **세후 추정**은 한계세율 + 지방소득세 10%의 단순 추정치입니다. 근로소득
  누진세 구조, 다른 소득과의 합산, 각종 공제는 반영하지 않습니다.
- **분량별 예상 세금**은 회사가 원천징수한 총 세금을 주식수 비율로 안분한
  값입니다. 실제로는 지급 시점에 일괄 원천징수되며 분량별로 따로 부과되지
  않습니다. 표시는 "이 분량에 대응되는 세금 부담은 얼마인가"의 회계적
  안분일 뿐입니다.
- **파운드리·LSI 분배 방식**: 파운드리·LSI는 사업부 재원(60%)에는
  적자 시 비참여. 부문 재원(40%)에서 메모리 대비 2026년 1.0배(동일),
  2027년부터 0.6배로 차등 분배됩니다. 흑자로 전환되면 사업부 재원
  분배에도 참여하게 되지만, 이 계산기는 항상 적자 상태로 가정합니다.

실제 지급액은 본 추정과 다를 수 있으며 회사 공식 안내 및 세무 신고 결과에
따릅니다.
