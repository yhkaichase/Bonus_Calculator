# 삼성전자 성과급 계산기 (DS 특별경영성과급 + OPI)

2026년 5월 20일 삼성전자 노사 잠정합의서에 기반한 비공식 참고용
1인당 예상 수령액 계산기입니다.

## 프로젝트 구조

```
.
├── index.html                      # 계산기 페이지
├── data.json                       # 자사주 환산 주가 데이터 (자동 갱신)
├── scripts/
│   └── update_data.py              # 다중 소스 종가 fetcher (Naver → Stooq → Yahoo 폴백)
├── .github/
│   └── workflows/
│       └── update-data.yml         # 평일 18:30 KST 자동 실행, 모든 소스 실패 시 이슈 자동 생성
├── .gitignore
└── README.md
```

## 사용자 입력

- **사업부** — 메모리 / 공통조직 / 파운드리·LSI
- **직급** — CL1~4. CL4만 보정 계수가 실제 적용됨
- **보정 계수** — ×1.0 / ×1.1 / ×1.2 / ×1.4 중 선택. CL4 한정.
  **특별경영성과급(부문+사업부)과 OPI 모두에 적용**. 라벨에 등급명을
  표기하지 않으므로 화면 캡처로 봐도 본인 정보를 추정하기 어렵습니다.
- **계약연봉액 + 업무 성과급** — 분배 기준 (raw 값, 보정 배수는 별도)
- **적용 연도** — 2026~2035. 임계값과 파운드리·LSI 가중치를 결정
- **DS 영업이익** — 시나리오 슬라이더 (조원 단위)
- **자사주 환산 주가** — 기본값은 `data.json`에서 자동 갱신

세금·건강보험은 입력 연봉과 성과급을 합산해 **누진세표(2026 귀속)**
및 **건보료율(2026 적용)**로 자동 산출됩니다. 별도 입력 없음.

## 계산 공식

### 1) 풀 분배 (특별경영성과급)

```
풀                  = DS 영업이익 × 10.5%
부문 풀             = 풀 × 40%        (DS 전체 가중 분배)
사업부 풀           = 풀 × 60%        (메모리·공통만, 적자 사업부 비참여)

메모리   X_m        = 부문 풀   / (N_m + 0.7·N_c + dlw·N_l)
메모리   Y_m        = 사업부 풀 / (N_m + 0.7·N_c)
공통     부문/사업부 = 0.7 × (X_m, Y_m)
파운드리·LSI 부문   = dlw × X_m       (dlw: 2026 = 1.0, 2027~ = 0.6)
파운드리·LSI 사업부 = 0               (적자 시 사업부 풀 비참여)
```

여기서 `dlw`는 적자 사업부 가중치, 1인당 금액은 각 사업부 기준치에
`salaryFactor = 본인연봉 / 평균연봉(8,700만)` 을 곱한 값입니다.

### 2) OPI

```
OPI 1인당           = 본인 연봉 × 50% × 보정계수
보정계수            = CL4 한정 (×1.0 / ×1.1 / ×1.2 / ×1.4)
                      CL1~3 는 항상 ×1.0
```

OPI는 영업이익에 비례하지 않는 **상한 가정** (단순화). 임계값 미달
시에도 지급됩니다.

### 3) 세후·자사주

```
성과급 세금         = tax(연봉 + 성과급) − tax(연봉)
                      (누진세표 기반, 구간 가로지름을 정확히 반영)
지방소득세          = 소득세 × 10%
성과급 세후         = 성과급 세전 − 성과급 세금
자사주 수량         = 성과급 세후 ÷ 환산 주가
                      (1/3 즉시 매각 / 1/3 1년 제한 / 1/3 2년 제한)

성과급 추가 건보료  = ins(연봉+성과급) − ins(연봉)
                      (건보 3.595% + 장기요양 13.14%, 보수월액 상한 고려)
실수령              = 성과급 세후 − 추가 건보료
                      (회사 자사주 처리는 세금만 차감한 "세후" 기준이고,
                       건보료는 본인부담 월급에서 별도 차감)
```

임계값 미달 시 특별경영성과급은 0이며, OPI는 영업이익과 무관하게 연봉
기반으로 항상 지급됩니다.

## 검증

300조 OP · 2026년 · 평균 연봉(8,700만) · 보정 ×1.0 시나리오:

| 사업부               | 부문 몫 | 사업부 몫 |   OPI | 합계  |
| :------------------- | ------: | --------: | ----: | ----: |
| 메모리               |  1.83억 |    3.86억 | 0.43억 | 6.12억 |
| 공통조직             |  1.28억 |    2.70억 | 0.43억 | 4.41억 |
| 파운드리·LSI (2026)  |  1.83억 |    0.00억 | 0.43억 | 2.26억 |

- 공통/메모리 비율: 부문 1.28/1.83 = **0.700** ✓ / 사업부 2.70/3.86 = **0.700** ✓
- 부산일보 보도("메모리 약 6억, OPI 포함")와 합계 6.12억이 근사 일치 ✓
- 파운드리·LSI 1.83 vs 보도 1.6: 부문 재원이 가중 분배되므로 메모리·
  파운드리가 같은 비율을 받음 (보도는 단순 78,000명 균등 분배 가정)

**2027년부터 파운드리·LSI 부문 가중치 0.6 적용**:

| 사업부               | 부문 몫 | 사업부 몫 |   OPI | 합계  |
| :------------------- | ------: | --------: | ----: | ----: |
| 메모리               |  2.07억 |    3.86억 | 0.43억 | 6.36억 |
| 공통조직             |  1.45억 |    2.70억 | 0.43억 | 4.58억 |
| 파운드리·LSI (2027~) |  1.24억 |    0.00억 | 0.43억 | 1.67억 |

LSI 페널티 분이 메모리·공통의 부문 몫으로 재분배되어 X_m이 1.83 → 2.07억
으로 증가하는 구조입니다.

**CL4 보정 ×1.4 적용 시** 모든 구성요소가 1.4배가 되어, 2026 메모리 합계
6.12 → 8.57억으로 상승합니다.

본인 연봉이 평균보다 높으면 `salaryFactor`로 부문·사업부 몫이 비례
증가합니다 (예: 연봉 1억이면 부문·사업부 몫 각각 × 1.149).

## 자동 주가 갱신

`data.json`은 GitHub Actions(`平日 18:30 KST` 스케줄)에서 자동
갱신됩니다. 단일 소스 의존을 피하기 위해 4단계 폴백 체인을 사용합니다:

1. **Naver Finance siseJson** (일봉) — 가장 정확
2. **Naver Finance polling** (실시간 시세)
3. **Stooq CSV**
4. **Yahoo Finance** (마지막 폴백, 클라우드 IP 차단 빈번)

모든 소스가 실패하면 워크플로가 자동으로 `data-update-failure` 라벨이
달린 이슈를 생성합니다 (같은 날 중복 생성 방지).

## 유의 (반드시 참고)

이 계산기는 공개 보도를 바탕으로 한 비공식 추정 도구이며, 다음 한계가 있습니다.

- **OPI 산정 모델**은 "입력 연봉 × 50%"의 상한 가정으로 단순화되어
  있으며, 영업이익에 비례하지 않습니다. 풀(영업이익 × 1.5%)과 일치
  하려면 DS 영업이익이 약 220조 이상이어야 하므로 저영업이익
  시나리오에서는 실제 OPI보다 과대 추정될 수 있습니다.
- **보정 계수 적용 범위** — 본 모델은 보정 계수를 특별경영성과급
  (부문 + 사업부)과 OPI에 모두 적용합니다. 공식 합의서의 적용 범위가
  다르면 모델 수정이 필요합니다.
- **사업부 인원수**는 부산일보 보도의 추산치(메모리 28천 / 공통 30천 /
  파운드리·LSI 20천 / DS 합계 78천)이며 공식 수치가 아닙니다. 코드 상단 `HC`
  객체를 수정해 본인이 아는 값으로 바꿔 사용할 수 있습니다.
- **평균 연봉 8,700만**은 2025 사업보고서 총보수 1.58억의 약 55%
  (OPI·TAI·시간외·복리후생 제외) 추정치입니다.
- **임계값 조건**은 특별경영성과급에만 적용됩니다. OPI 는 영업이익과
  무관하게 연봉 기반으로 지급되는 것으로 가정합니다. 공식 합의서에서
  OPI 도 임계값 조건이 걸려 있다면 모델 수정이 필요합니다.
- **세금**은 누진세표(소득세법 제55조) + 지방소득세 10%로 자동 산출
  되며, 인적공제·세액공제·다른 소득과의 합산은 반영하지 않은
  간이 추정치입니다.
- **건강보험·장기요양**은 직장가입자 본인부담률(건보 3.595%, 장기요양
  13.14%)을 연 환산한 추정치이며, 보수월액 상한을 고려합니다. 실제
  성과급 발생 월의 정산 처리와는 차이가 있을 수 있습니다.
- **자사주는 세금만 차감한 세후 금액 기준으로 환산**됩니다 (회사
  처리 방식). 건보료는 별도 월급에서 차감되므로 "실수령" 표시값과
  자사주 환산 기준 금액이 다를 수 있습니다.
- **파운드리·LSI 분배 방식**: 사업부 재원(60%)에는 적자 시 비참여.
  부문 재원(40%)에서 메모리 대비 2026년 1.0배, 2027년부터 0.6배로
  차등 분배됩니다. 흑자로 전환되면 사업부 재원 분배에도 참여하게
  되지만, 이 계산기는 항상 적자 상태로 가정합니다.

실제 지급액은 본 추정과 다를 수 있으며 회사 공식 안내 및 세무 신고 결과에
따릅니다.
